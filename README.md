## __5.ナビゲーションメッシュ__　　　　　　　　
3ds MAXでマップを作成したデータをレベルで読み込ませ。
地形データを元に経路探索をして、マップ上の穴を自動で避けるようにプレイヤーに向かって追いかけるように作りました。


## __6.ゲームでこだわった部分__　            
・ゲームのバランス
